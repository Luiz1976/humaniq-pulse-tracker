# Manual do Acelerador de SEO HumaniQ

Foi adicionada uma nova funcionalidade de **Aceleração de Ranking SEO** na seção de Conteúdo do Website.

## Como acessar
1. No painel principal (Dashboard), localize o botão **"SEO Booster"** na barra superior de navegação.
2. Clique no ícone de "gráfico subindo" (Trending Up).
3. O painel se abrirá diretamente na tela principal.

## Funcionalidades

### 1. Monitoramento de Páginas
O sistema monitora automaticamente as 10 páginas estratégicas definidas:
- Home
- NR01 (Pilar e Software)
- Riscos Psicossociais
- Blog e Artigos Específicos

### 2. Ciclo de Indexação (Ping)
Ao clicar em **"Iniciar Ciclo"**, o sistema visita cada uma das páginas. Isso gera:
- Um sinal de tráfego/atividade para os buscadores.
- Uma verificação de disponibilidade (Status 200).
- Renovação do cache do navegador.

### 3. Modo Automático (Alta Frequência)
Ative o botão **"Modo Automático"** para que o sistema execute o ciclo de otimização a cada **60 segundos** de forma contínua enquanto a aba estiver aberta.
Isso atende à sua solicitação de "pesquisar com alta frequência diretamente por essas páginas".

> **Nota:** Para melhores resultados, mantenha a aba aberta em segundo plano com o Modo Automático ativado durante períodos estratégicos do dia.

## Estratégia de Indexação
Esta ferramenta ajuda a manter as páginas "quentes" e monitoradas. Para posicionamento definitivo na primeira página, continue utilizando a geração de conteúdo de alta qualidade através do sistema de Blog integrado.
